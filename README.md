## Features

Provides a Link object with the following options:
  - Anchor link
  - Internal Link
  - External Link
  - Email
  - Telephone
  - File
  - Image
  - Link target (_blank etc)

## Installation

Add the following to your composer.json file and run /dev/buid?flush=all

```json
{  
    "require": {  
        "cyber-duck/silverstripe-linkitemfield": "4.0.*"
    }
}
```
